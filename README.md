# chatgpt_Mobile_Adoption

# overview
I have recently joined the ChatGPT India mobile team. In your first week, your manager has given you a project to work on: increase the usage of voice input on the ChatGPT mobile app.

While voice input is available on the app, very few users in India seem to be using it. Your manager believes that voice input could be a powerful tool, especially in a mobile-first country like India, but adoption has remained low. Your first task is to understand the landscape and figure out why.

To get started, your manager has asked you to complete the following in the first 2 weeks:

Understand voice input usage trends in India: how do Indian users interact with voice features on other apps (e.g. Google Assistant, WhatsApp voice notes, Alexa)?

Analyse the ChatGPT mobile experience: where is voice input surfaced? Is it discoverable, intuitive, and contextually helpful?

Identify 2–3 competing or analogous products and evaluate how they integrate voice interactions

Understand why increasing voice usage could be valuable for ChatGPT — and outline potential product outcomes
